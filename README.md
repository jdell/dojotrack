# DojoTrack

Martial arts club management, simplified. Classes, belt progression, payments — mobile-first.

## Status

Planning phase. See [docs/](docs/) for the full product strategy.

## Docs

- [Product plan](docs/product-plan.pdf) — names, market analysis, pricing, phased roadmap, wireframes, go/no-go checklist

## Tech stack

- Next.js 16 (App Router)
- Supabase (PostgreSQL, Auth, Realtime, Storage)
- Stripe (subscriptions, exam fees)
- WhatsApp Business API (notifications)
- Vercel (hosting)
- Prisma (ORM)
- Tailwind CSS + shadcn/ui

## Key feature

Belt progression tracker — students see their journey from white to black belt with a visual requirements checklist. Supports BJJ, Karate, Judo, Taekwondo, and custom ranking systems.
